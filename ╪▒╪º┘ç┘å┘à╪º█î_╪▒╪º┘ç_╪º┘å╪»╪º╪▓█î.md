# راهنمای راه‌اندازی PLAY Dodi — نسخه مستقل

## مرحله ۱ — ساخت دیتابیس در Supabase (رایگان)

1. به [supabase.com](https://supabase.com) بروید و ثبت‌نام کنید (رایگان)
2. روی **"New Project"** کلیک کنید
3. نام پروژه: `play-dodi`، یه رمز قوی انتخاب کنید، Region: `West EU (Ireland)`
4. صبر کنید تا پروژه آماده شود (~1 دقیقه)
5. از منوی چپ به **SQL Editor** بروید
6. محتوای فایل `SUPABASE_SETUP.sql` را کپی کنید و **Run** بزنید
7. از منوی چپ به **Project Settings → API** بروید
8. دو مقدار را کپی کنید:
   - **Project URL** (شبیه `https://xxxxx.supabase.co`)
   - **anon public** key

## مرحله ۲ — تنظیم پروژه

1. فایل `.env.example` را کپی کنید و نام آن را `.env` بگذارید
2. مقادیر را پر کنید:
```
VITE_SUPABASE_URL=https://xxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGci...
```

## مرحله ۳ — نصب و اجرا

```bash
npm install
npm run dev
```
برنامه روی `http://localhost:5173` باز می‌شود.

## مرحله ۴ — انتقال داده‌های قدیمی

1. وارد برنامه شوید
2. به بخش **Rapports** بروید
3. روی دکمه **Restaurer** کلیک کنید
4. فایل `backup_playdodi_2026-06-12_22-16.json` را انتخاب کنید
5. تمام! همه داده‌هایتان برگشت ✅

## مرحله ۵ — انتشار آنلاین (رایگان)

### با Vercel:
1. به [vercel.com](https://vercel.com) بروید
2. پروژه را از GitHub import کنید
3. در Environment Variables مقادیر Supabase را اضافه کنید
4. Deploy!

### با Netlify:
1. به [netlify.com](https://netlify.com) بروید
2. فولدر پروژه را drag & drop کنید
3. Environment Variables را تنظیم کنید

## نکات مهم
- برنامه کاملاً رایگان است (Supabase رایگان تا 500MB)
- هیچ نام base44 نمایش داده نمی‌شود
- شریک شما می‌تواند با همان کد عددی وارد شود
- Backup/Restore دقیقاً مثل قبل کار می‌کند
